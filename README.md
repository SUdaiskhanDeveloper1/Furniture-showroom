This project is furniture showroom management system that allows users to browse, manage, and purchase furniture items online. It provides a user-friendly interface for customers to explore various furniture collections, view detailed product information, and make secure transactions. The system also includes features for inventory management, order tracking, and customer support to enhance the overall shopping experience.
